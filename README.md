CUTM Result Portal

The CUTM Result Portal is a comprehensive web application that enables students and administrators to easily access, manage, and update academic results. It provides a secure, user-friendly platform for handling student performance data efficiently.

🚀 Live Demo

[https://cutm-result-portal-efhz.onrender.com]

✨ Key Features

Student Result Access: Students can instantly view their results by entering their registration number and selecting the semester.
Admin Dashboard: Administrators can log in to manage student profiles, update academic records, and oversee result data.
Responsive Design: Built with Bootstrap 5, ensuring seamless access across desktops, tablets, and mobile devices.
Dynamic Result Loading: Results are fetched and displayed in real time based on user input.
Secure Authentication: Robust login and authorization system to safeguard sensitive academic information.

🛠️ Technologies Used
Backend

Flask – Lightweight Python web framework for building the server-side logic.
MongoDB – NoSQL database for handling unstructured student input and additional data.
dotenv – For managing environment variables securely.
PyMongo – Python driver for MongoDB interactions.
Datetime – For handling date and time operations.
Jinja2 – Templating engine for rendering dynamic HTML content.

Frontend

HTML5 & CSS3 – Core structure and styling.
Bootstrap 5 – Responsive, mobile-first design.
JavaScript – Enhancing interactivity and smooth user experience.
